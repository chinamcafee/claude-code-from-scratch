# 第 9 章：终端输出层总览

`src/ui.ts` 虽然不是系统里最复杂的模块，但它把普通输出、工具输出、spinner、计划审批展示、子代理提示和工具摘要都放在了同一个文件里。

所以这一章拆成 3 个独立文件，按顺序把终端交互层搭起来。

## 本章路线

1. [09-ui-01-basic-output.md](./09-ui-01-basic-output.md)：先搭好基础打印函数和工具摘要辅助函数。
2. [09-ui-02-spinner.md](./09-ui-02-spinner.md)：补上 spinner 动画。
3. [09-ui-03-plan-subagent-final.md](./09-ui-03-plan-subagent-final.md)：补上计划审批和子代理提示，收口到最终版。

## 这一章做完后的验收目标

1. `diff -u "$REFERENCE_REPO/src/ui.ts" "$TARGET_REPO/src/ui.ts"` 没有输出。
2. 你可以输出欢迎语、工具调用摘要、错误和普通提示。
3. 你可以运行 spinner、计划审批展示和子代理起止提示。


从这里开始：[09-ui-01-basic-output.md](./09-ui-01-basic-output.md)
